Binho Python Package Documentation
================================================

This Python package contains everything needed to work with Binho USB host adapters at any level of access.
The package contains the lowest level API wrapper which can be used to send basic commands to the device, as well as
high-level tools such as a command-line-interface to easily send and receive data over the supported protocols.
The package also contains and interactive shell which provides an easy way to manually interact with the device.

This library is an entire redesign of our initial binhoHostAdapter python library which provides significantly more features
and is must easier to use. This package is under constant development with numerous features planned for release soon.
